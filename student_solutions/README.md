# Student Solutions

This folder contains examples of student solutions, in hopes of creating a diverse repository of possible approaches. Solution contributions are welcome from everyone!

For homework assignments with test cases, please ensure your solution passes all test cases before contributing. For homework assignments without test cases, please contribute your best effort.

## How to Contribute

1. Create a fork of this repository.

2. COPY out the `problems.py` file (or `problems_*.py` files) from all the folders of all homeworks you wish to submit, and place them into the following path with the following naming convention:

    ```
    student_solutions/<HW>/<PROBLEMS-FILENAME>_<YOUR-GITHUB-USERNAME>.py
    ```

    For example, if I (`ML72` on GitHub) want to submit my `HW1`, `HW2`, and `HW4` solutions, I would create the following files:

    ```
    student_solutions/HW1/problems_ML72.py

    student_solutions/HW2/problems_numpy_ML72.py
    student_solutions/HW2/problems_pandas_ML72.py

    student_solutions/HW4/problems_clustering_ML72.py
    student_solutions/HW4/problems_regression_ML72.py
    ```

    Feel free to submit as many or as little of the homework assignments as you'd like!

3. Create a pull request to merge into this repository!
